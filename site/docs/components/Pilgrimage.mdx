# Pilgrimage 组件

## 说明

用于展示“圣地巡礼”实景与动画截图的对比卡片，支持按动画系列分组并通过下拉选择切换系列。

## Props

| 名称      | 类型                     | required | 默认值 | 说明                   |
| --------- | ------------------------ | -------- | ------ | ---------------------- |
| animes    | `PilgrimageAnimeGroup[]` | true     | -      | 分组数据（按动画系列） |
| className | string                   | false    | -      | 组件根节点额外类名     |

### 类型定义

```ts
export interface PilgrimageItem {
  realImageUrl: string;
  animeImageUrl: string;
  description?: string;
  shootDate?: string;
  location?: string;
}

export interface PilgrimageAnimeGroup {
  anime: string; // 动画标题/系列名
  items: PilgrimageItem[];
}
```

## 使用

```ts
import { Pilgrimage } from '@yixiaojiu/kasumi';

const App = () => (
  <Pilgrimage
    animes={[
      {
        anime: 'Anime 1',
        items: [
          {
            realImageUrl: 'https://images.unsplash.com/photo-1549692520-acc6669e2f0c?q=80&w=1280&auto=format&fit=crop',
            animeImageUrl: 'https://images.unsplash.com/photo-1549692520-acc6669e2f0c?q=80&w=1280&auto=format&fit=crop',
            shootDate: '2023-04-10',
            location: '新宿区',
            description: '新宿御苑入口附近取景',
          },
        ],
      },
    ]}
  />
);
```
